<ol dir="auto">
<li>Récupérer le repository :</li>
</ol>
<div dir="auto"><pre>git clone https://github.com/ainemehdi6/ITRomTest.git</pre></div>
<ol start="2" dir="auto">
<li>Se positionner dans le dossier:</li>
</ol>
<div dir="auto"><pre><span>cd</span> ITRomTest-main</pre></div>
<ol start="3" dir="auto">
<li>Installer les dépendances :</li>
</ol>
<div dir="auto"><pre>composer install</pre></div>
<ol start="4" dir="auto">
<li>Base de donnée :
N'oubliez pas de modifier le ".env" pour l'adapter à votre base de donnée.</li>
</ol>
<div dir="auto"><pre><span><span>#</span> DATABASE_URL="sqlite:///%kernel.project_dir%/var/data.db"</span>

<p dir="auto">Vous devez créer une base de donnée en local avec le nom : 'it_room'
Ensuite exécuter :</p>
<div dir="auto"><pre>php bin/console doctrine:schema:update --force</pre></div>
<ol start="5" dir="auto">
<li>Lancer le projet :</li>
</ol>
<div dir="auto"><pre>symfony server:start</pre></div>
